how to run the code :
1.) first open vscode .
2.) then go to open folder and select file "SHOPKEEPER PORTAL"
3.) then open terminal on vscode.
4.) then click to change the directory to Stock-Flow:  cd Stock-Flow
5.) then install npm i.e : npm install.
6.) then run: npm run dev
7.) then run: npm run dev:client
finally the website link comes that run it.
